函数要点

  • 函数名和变量名使用蛇形命名法(snake case)，例如 fn add_two() -> {}
  • 函数的位置可以随便放，Rust 不关心我们在哪里定义了函数，只要有定义即可
  • 每个函数参数都需要标注类型

函数参数

Rust 是静态类型语言，因此需要你为每一个函数参数都标识出它的具体类型

Rust 中的特殊返回类型

无返回值()

  • 函数没有返回值，那么返回一个 ()
  • 通过 ; 结尾的语句返回一个 ()
